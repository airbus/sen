Detailed Description

template<typename T, typename MutexType = std::mutex>
struct sen::Guarded< T, MutexType >::TemporaryAccessToken

Access token that enables the user to interact with the protected object while simultaneously holding a lock to ensure access are correctly synchronized.

Note: The TemporaryAccessToken should only be held as a temporary or stored for short periods of time, to ensure that the underlying object can be accessed by other threads.
