# Crash record test

This directory has a simple recording file that is not closed gracefully to test the resilience of the recorder.

The recording can be generated by running sen with the crash_record.yaml:

1. Put the python script in your environment:
`export PYTHONPATH=$PYTHONPATH:<path_of_this_repo>/components/recorder/test/data`

2. Run the code and let it finish:
`cd <path_of_this_repo>/components/recorder/test/data && sen run crash_record.yaml`

3. A `crash_recording` directory should be generated in your working directory
